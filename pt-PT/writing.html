Notas

Peças curtas. Coisas que quero lembrar, na maioria. Algumas sobre marcas, algumas sobre jogos, algumas sobre os dois.

Porque reconstruo em vez de auditar.
Auditorias são teatro. Uma reconstrução é compromisso — confias-me a marca, confio-te o resultado. A primeira hora de uma auditoria é igual à de uma reconstrução. As cem seguintes não são.
O que o Pokémon FireRed me ensinou sobre menus.
O menu START — POKEDEX / POKEMON / BAG / [JOGADOR] / SAVE / OPTION / EXIT — é um exemplo perfeito de restraint. Sete itens, sem ícones, sem decorações. Sabes sempre onde estás. Metade dos dashboards SaaS que reviso podia aprender com isso.
O perfil de marca é metade do trabalho.
Quando já extraí voz, audiência, hard-nos e o one-liner de uma marca, o site quase se escreve sozinho. A lista de páginas, a paleta, o registo da copy — tudo sai do perfil. O HTML é a parte fácil.
